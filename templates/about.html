About This Project

This web-based application detects whether a news article is Fake or Real using Machine Learning and Natural Language Processing (NLP).

Project Information

Project Title: Fake News Detection Using Machine Learning

Dataset: ISOT Fake News Dataset

Project Goal

To identify whether a news article is Fake or Real using Machine Learning techniques and help reduce the spread of misinformation.

Technologies Used

  • Python
  • Flask
  • Machine Learning
  • Natural Language Processing (NLP)
  • TF-IDF Vectorizer
  • Linear SVM
  • Scikit-learn
  • NLTK
